# Methods for preprocessing and segmentation of 64mT MRI of infants

## Contents

1. Scripts - bash and python scripts for executing super-resolution reconstruction
             on HPC systems. 
1. Deconvolution - docker build files for creating the deconvolution tools. 
1. PSFEstimation - tools for estimating point spread functions based on phantom data
1. DistortionFieldEstimation - tools for estimating unwarping fields
